如果遇到了Git提示更换了服务器key失败的话

```bash
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
@    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
```

可以尝试着使用下面的命令解决：

```bash
ssh-keygen -R YourIPorDomainName
# 例如 ssh-keygen -R github.com
```
